# Meet Amber — Digital Cover Letter

A single-page digital cover letter created for my application for a Front-End Developer position at Corporate Tools.

Instead of submitting a traditional cover letter, I wanted to create something that reflects both my personality and my interest in front-end development.

## About the Project

This project introduces who I am, why I chose software development, and why I am interested in joining Corporate Tools.

The goal was to create a cover letter that feels professional while still sounding like me.

The page highlights:

- Why I chose software development
- Why I am interested in Corporate Tools
- How I approach learning and teamwork
- A little about the person behind the code

## Built With

- HTML
- Tailwind CSS
- Responsive design

The project was intentionally built as a **single HTML file** so it could be submitted directly as a digital cover letter with my job application.

## Design

I chose a clean, modern design with a dark color palette and simple typography so the focus stays on the content.

The design is intentionally different from a traditional résumé or portfolio. It is meant to feel more like a personal introduction.

The page is also responsive so it can be viewed comfortably on different screen sizes.

## What I Learned

This project gave me an opportunity to practice:

- Building a responsive layout with Tailwind CSS
- Using utility classes to create a consistent design
- Creating clear visual hierarchy with typography and spacing
- Thinking about accessibility and semantic HTML
- Designing around a specific user and purpose
- Turning a traditional cover letter into an interactive digital experience

## Project Structure

The project is intentionally simple:

```text
digital-cover-letter/
├── index.html
└── README.md
```

## Why I Built It

Corporate Tools asks applicants to submit a cover letter rather than a traditional résumé.

Since I am applying for a Front-End Developer position, I decided to build mine.

It gave me an opportunity to introduce myself while also demonstrating some of the skills I hope to bring to the role.
